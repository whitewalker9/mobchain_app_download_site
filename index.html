Mobile Theft Detection using Blockchain

Nowadays, mobile theft has increased drastically all over the world. The prevention of mobile theft is essential to enhance device security. A large number of devices have been stolen for the lack of essential foundation and administration of a secure platform. Several existing systems were developed to protect the mobile from unauthorized access.

But, they suffer from various limitations such as data security, the possibility of cybercriminals activities, leakage of personal information, and centralized System. We propose a mobile theft detection framework using a decentralized and secure platform to increase the security level of the mobile phones.

Blockchain and smart contracts are used to provide the security of the stored data on the ledger and authenticate a genuine user automatically with greater accuracy. The proposed framework can provide mobile security and owner’s privacy. The implementation of blockchain makes it easy to check a mobiles’s real history.
